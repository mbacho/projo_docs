Overview

Problem statement

There is no effective method to analyze and determine the effectiveness and impact academic website's system of websites

Justification

Provision of information about the effectiveness and impact of websites will enable website administrators to make more informed decisions. For example, they would know when to improve on their website or policies governing the same.

Literature review

Website analysis methods

  • Qualitative methods
    Content analysis, focus groups, interviews, referrer analysis, user feedback and audience analysis
  • Quantitative methods
    Website log analysis, bibliometrics, scientometrics and webometrics

Webometric Ranking of Universities

  • Done biannually (Jan, July) by CSIC
  • Ranks over 20,000 universities
  • Metrics shown : web presence, web impact, openness and excellence

Ranking analytics analytics used


Visibility 50% (link visibility 20% and G-factor 30%), Size (20%) Rich Files (15%), and Scholar (5%) and Scimago (10%)

Resources used

  • Linux operating system
    This is the operating system of choice for the developer.
  • Python programming language
    The system was written using Django web application framework which used python programming language.
  • Postgresql DBMS
    The database was used as the application data store. It stored all the data in the system except the crawl data.

    • Resources cont...

  • Mongo DB
    The database was used to store all the crawl data generated by the system. This database was preferred since it had the ability to handle millions of records easily and thus able to scale. It also was able to work with missing data.
  • Git
    It was used to version the system's source code and documentation.
  • Chromium and Firefox Web browser
    The two web browsers were used to test and use the system's web interface.

    • Resources cont....

  • Rabbit MQ
    It was used as a message broker between the crawler server and the rest of the application. It mediated communication between the two thus decoupling them.
  • Hardware resources
    Laptop PC (Intel core i5 processor, over 4gb ram and over 2 gb free harddisk space)
    Internet connection
